那么，新的 Runes API 有什么特别之处呢？简单来说，它的设计目的是让状态的创建和管理变得更加简单。起初我有些怀疑，但在使用之后，我被它打动了！它更加直观，让编写整洁和可维护的代码变得更加容易。

我期待在 [SticAi](https://sticai.com) 和 [Uiino](https://uiino.com) 等产品中使用这个新 API，以提高代码库的性能和可维护性。
